### 1. Acode (Android via F-Droid)
Acode adalah editor kode ringan untuk Android yang mendukung HTML, CSS, dan JavaScript.

**Langkah instalasi:**
1. Buka aplikasi **F-Droid** di perangkat Android.
2. Cari **Acode** di kolom pencarian.
3. Pilih aplikasi **Acode** lalu tekan **Install**.
4. Setelah terpasang, buka Acode dan mulai membuat file HTML/CSS/JS.

---

### 2. VSCodium (Windows)
VSCodium adalah versi open-source dari Visual Studio Code tanpa telemetry.

**Langkah instalasi:**
1. Buka situs resmi [VSCodium](https://vscodium.com).
2. Unduh installer untuk **Windows (.exe)**.
3. Jalankan installer dan ikuti langkah-langkah instalasi.
4. Setelah selesai, buka VSCodium dan mulai coding dengan membuat project baru.

---
